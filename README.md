# Noa Insure

이 프로젝트는 바이브 코딩(Vibe Coding) 원칙을 준수하여 작성되는 웹/앱 프로젝트입니다.

## 프로젝트 구조 (Project Structure)

- `/.gemini/`: AI 어시스턴트 및 프로젝트 코딩 컨벤션/규칙 문서
- `/docs/`: 기획서, API 명세, 회의록 등 프로젝트 기록물
- `/front/`: 프론트엔드(Web/App) 소스 코드
- `/back/`: 백엔드(API, DB 등) 소스 코드

## 개발 원칙

자세한 개발 규칙은 `.gemini/rules.md`를 참고해 주세요.
폴더 깊이를 최소화(Anti-gravity)하고 프론트엔드와 백엔드를 철저히 분리하여 관리합니다.
