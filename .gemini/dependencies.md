# Dependency Rules

## 1. 불필요한 라이브러리 사용 금지 (No Unnecessary Libraries)

- 외부 의존성은 최소한으로 유지하세요.
- 바닐라(Vanilla) 언어 기능이나 이미 프로젝트에 포함된 기본 도구로 해결 가능한 문제는 새로운 라이브러리를 추가하지 않고 해결하세요.
- 라이브러리 추가가 반드시 필요한 경우, 번들 사이즈와 유지보수성을 신중히 검토하세요.

## [프로젝트 적용 사례]

- **대시보드 통계 차트**: Chart.js 등 무거운 외부 시각화 라이브러리를 사용하지 않고, 순수 HTML/CSS Flexbox와 Vanilla JS를 활용해 상태 비율 차트(Progress Bar)를 구현했습니다.
- **프론트엔드 환경**: React나 Vue 없이 순수 Vanilla JS 모듈(`type="module"`)만으로 컴포넌트 간 역할 분리와 상태 관리를 완벽하게 구현했습니다.
