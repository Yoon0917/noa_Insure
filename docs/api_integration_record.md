# 외부 통신 연동 기록 (API Integration Record)

## 2026-04-25: 실시간 외부 API (업비트) 연동 테스트 완료

### 📌 목적

관제 플랫폼의 프론트엔드(`api.js`)가 실제 외부 서버와 정상적으로 데이터를 주고받을 수 있는지 시스템 아키텍처의 건전성을 검증.

### ⚙️ 구현 내용

- **대상 API**: 업비트(Upbit) Ticker API (`https://api.upbit.com/v1/ticker?markets=KRW-BTC`)
- **처리 방식**: `fetch` 모듈을 활용하여 데이터를 수신하고, 관제 대시보드 리스트 규격에 맞게 변환(Mapping)하여 최상단에 렌더링.
- **UX 연동**: 수신된 실제 JSON 전문(Payload)이 모달창에서 정상적으로 표시되며, 필요한 경우 마스킹 처리가 호환되는지 확인 완료.

### ✅ 결과 및 의의

프론트엔드-백엔드 의존성 분리 원칙에 따라 `api.js` 파일이 네트워크 통신 책임을 완벽히 전담함을 증명했습니다. 향후 사내 시스템(Node.js/Express 등) API가 구축되면, URL과 응답 처리 로직만 교체하여 즉각적으로 실 서비스에 도입할 수 있는 상태가 되었습니다.
